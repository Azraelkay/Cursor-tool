# 🛠️ Cursor Tools - Cursor环境工具

一个专为Cursor IDE设计的环境清理工具，帮助重置Cursor环境并提升使用体验。

## 🎯 工具简介

Cursor Tools是一个基于积分制的Cursor编辑器环境工具，主要功能：
- 🧹 **环境清理** - 清理Cursor配置、缓存和用户数据
- 🔄 **环境重置** - 重置设备ID和用户标识
- 💎 **积分系统** - 新用户赠送30积分，支持多种获取方式
- 🔄 **自动更新** - 程序可联网自动检查更新

- 🎁 **新用户奖励** - 初始获得30积分

- ⭐ **GitHub Star** - 给项目点Star获得10积分（一次性）
- 📅 **每日签到** - 每天签到获得1积分

- 🧹 **清理操作** - 每次清理消耗10积分

## 🚀 使用步骤

### 📥 下载安装
1. 从 [Releases](https://github.com/your-username/cursor-tools/releases) 下载最新版本
2. 解压到任意目录
3. **以管理员身份运行** `Cursor Tools.exe`

### 📋 使用前准备
1. **完全卸载Cursor**
   - 推荐使用 [Geek Uninstaller](https://geekuninstaller.com/) 彻底卸载
   - 或使用系统自带的"添加或删除程序"
   - 确保清除所有残留文件和注册表项

2. **备份重要数据** - 备份重要的项目和配置文件
3. **管理员权限** - 右键选择"以管理员身份运行"

### 🎁 获取积分
1. **GitHub Star方式**：
   - 点击 "⭐ GitHub Star (10积分)" 按钮
   - 在打开的GitHub页面点击Star
   - 返回程序点击确认，获得10积分

2. **每日签到方式**：
   - 每天点击 "📅 每日签到 (1积分)" 按钮
   - 连续签到累积积分

### 🧹 开始清理
1. 确保有足够积分（≥10积分）
2. 点击 "🧹 清理Cursor (消耗10积分)" 按钮
3. 等待清理完成
4. 查看清理结果

## 📋 清理内容

- **配置文件** - Cursor用户配置和设置
- **缓存文件** - 应用缓存和临时文件
- **日志文件** - 运行日志和错误日志
- **用户数据** - 用户标识和设备信息
- **扩展数据** - 扩展相关的缓存数据
- **注册表项** - 相关的Windows注册表项

## 🔄 自动更新

- **启动检查** - 程序启动时自动检查更新
- **版本比较** - 智能比较本地和远程版本
- **一键下载** - 发现新版本可一键下载
- **自动安装** - 下载完成后自动安装并重启
